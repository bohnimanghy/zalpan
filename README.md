# Zalpan

Marketing site for **Zalpan** — a restaurant operating system (POS & billing,
kitchen display, inventory, QR ordering, CRM, vendor management, analytics and
AI forecasting). A [Bohniman](https://bohniman.com) product.

Built with Next.js 16 (App Router, Turbopack) + Tailwind CSS v4.

## Develop

```bash
npm install
npm run dev        # http://localhost:3000 (use -p 3001 if bohniman.ai is running)
```

## Build

```bash
npm run build
npm start
```
